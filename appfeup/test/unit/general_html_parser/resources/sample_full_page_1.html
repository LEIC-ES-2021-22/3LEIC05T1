Performance API (PAPI)

PAPI Download 


Installation: 
Uncompress to a directory and change to "src" directory
./configure 
make 
sudo make install 
sudo ldconfig 

Compiling tests code: 
make test 
cd ctests 
./zero. (run tests) 

To use PAPI: 
g++ -O2 file.cpp -o fileout -lpapi 

Installing papi-tools: sudo apt-get install papi-tools 
Check available counters: papi_avail -a

NOTE:

To be able to read performance counters on LINUX, you need to edit the following file:

/proc/sys/kernel/perf_event_paranoid 

and put -1 instead of 3.

sudo sh -c 'echo -1 >/proc/sys/kernel/perf_event_paranoid'
